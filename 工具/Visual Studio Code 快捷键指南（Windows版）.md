#### 通用操作

|快捷键|功能描述|
|---|---|
|Ctrl+Shift+P/F1|显示命令面板|
|Ctrl+P|快速打开（转到文件）|
|Ctrl+Shift+N|新建窗口/实例|
|Ctrl+Shift+W|关闭窗口/实例|
|Ctrl+,|用户设置|
|Ctrl+K Ctrl+S|键盘快捷键设置|
