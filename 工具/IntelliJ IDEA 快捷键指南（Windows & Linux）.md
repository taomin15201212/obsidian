快速打开文件名
Ctrl+Shift+N
#### 选中下一个相同的词

- Alt + j

#### 选中所有相同的词

- Ctrl + Shift + Alt + j

Ctrl + Y 删除光标所在行 或 删除选中的行 （必备）


#### 编辑操作

| --------------------- | ----------------------------------------- |
