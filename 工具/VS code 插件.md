### 一键启动

**Mac** `Control + Shift + Q`

**Win** `Shift + Alt + Q`

- Change Chapter 可以切换章节
- Change Dictionary 可以切换字典
- Start/Pause 可以开关插件，功能等价于一键启动快捷键
- Toggle Word Visibility 切换是否展示单词（默写模式）
- Toggle Read Only Mode 开关只读模式
- Toggle Chapter Cycle Mode 章节循环模式

命令面板快捷键  
Mac: `cmd + shift + p`  
Win: `ctrl + shift + p`
